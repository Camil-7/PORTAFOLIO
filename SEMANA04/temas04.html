Temas Aprendidos

HTML Básico

1. Introducción al HTML y estructura básica de un documento.
2. Etiquetas principales: <html>, <head>, <body>.
3. Formateo de texto con <p>, <h1>-<h6>, <strong>, <em>, <br>, <hr>.
4. Listas con <ul>, <ol>, <li>.
5. Imágenes con <img>, atributos src y alt.
6. Enlaces con <a>, atributos href y target.
7. Tablas con <table>, <tr>, <td>, <th>.
8. Formularios básicos: <form>, <input>, <label>.
9. Comentarios en HTML: <!-- Comentario -->.
10. Estructura semántica: <header>, <footer>, <main>, <section>, <article>.

HTML Avanzado

1. Multimedia: <audio> y <video>.
2. Metadatos con <meta>.
3. Formularios avanzados: <select>, <textarea>, checkbox, radio.
4. APIs HTML5: localStorage, geolocalización.
5. Elementos interactivos: <details> y <summary>.
6. Integración de JavaScript y CSS con <script> y <link>.
7. Buenas prácticas y accesibilidad.
8. SEO y estructura semántica.
9. Microdatos y datos estructurados.
10. Diseño responsivo y móvil.

CSS Básico

1. Introducción a CSS y cómo aplicarlo.
2. Selectores básicos: etiqueta, clase e ID.
3. Propiedades de color y fondo.
4. Modelo de caja (box model).
5. Posicionamiento en CSS.
6. Tipografía y texto.
7. Display y visibilidad.
8. Pseudoclases en CSS.
9. Unidades en CSS.
10. Selectores avanzados y agrupación.
