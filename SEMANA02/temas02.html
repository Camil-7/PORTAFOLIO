Temas Aprendidos
Visual Studio Code Editor de código fuente liviano, gratuito y multiplataforma. Permite trabajar con HTML, CSS, JS, terminal integrada y extensiones útiles.
Interfaz de Usuario Incluye el explorador de archivos, barra de estado, editor, terminal integrada y paleta de comandos.
Paleta de Comandos Acceso con Ctrl+Shift+P para ejecutar rápidamente funciones como "crear terminal", "comentar línea", o usar Docker.
Terminal Integrado Permite ejecutar scripts, comandos npm, git y más directamente desde VSC.
Emmet Herramienta que agiliza la escritura de código HTML/CSS mediante abreviaciones como div>ul>li*3 o .clase.
Atajos de Teclado Acciones como comentar líneas, duplicar líneas, abrir archivos o minimizar paneles con combinaciones como Ctrl + / o Ctrl + B.
Selectores y Pseudo-clases CSS Aplicación de estilos dinámicos como :hover, :first-child, :nth-child para mejorar la presentación y usabilidad del sitio web.
